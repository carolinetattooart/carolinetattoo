# Caroline Tattoo — Sitio Web

## Estructura de archivos

```
caroline-tattoo/
├── index.html          ← Página principal
├── styles.css          ← Estilos (estética premium/minimal)
├── main.js             ← Interactividad
├── assets/
│   ├── portfolio/      ← TUS FOTOS van aquí
│   │   ├── bw-01.jpg   (blackwork)
│   │   ├── bw-02.jpg
│   │   ├── bw-03.jpg
│   │   ├── lw-01.jpg   (linework)
│   │   ├── lw-02.jpg
│   │   ├── or-01.jpg   (ornamental)
│   │   ├── or-02.jpg
│   │   ├── mn-01.jpg   (minimalista)
│   │   └── mn-02.jpg
│   ├── logo.svg        ← Tu logo
│   └── hero.jpg        ← Foto principal del hero
└── README.md
```

## Cómo subir tus fotos

1. Nombra tus fotos exactamente como aparece arriba
2. Ponlas dentro de la carpeta `assets/portfolio/`
3. Para la foto del hero: `assets/hero.jpg`
4. Para el logo: `assets/logo.svg` o `.png`

## Pendientes de configurar

- [ ] Reemplazar URL de Calendly en `index.html` (línea con `data-url="https://calendly.com/caroline-tattoo/consulta"`)
- [ ] Agregar fotos reales al portafolio
- [ ] Subir foto del hero

- [ ] Conectar formulario de contacto (Formspree recomendado - gratis)
- [ ] Subir logo

## Activar GitHub Pages

1. Ve a tu repo en github.com
2. Settings → Pages
3. Source: Deploy from a branch
4. Branch: main / root
5. Save
6. En ~2 minutos estará en: `https://carolinetattooart.github.io/caroline-tattoo`
