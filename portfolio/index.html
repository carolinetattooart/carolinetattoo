About

Blackwork uses solid black ink to create dramatic, high-contrast designs. It is one of the most durable tattoo styles — black ink retains better than any color over time. Each piece is designed thinking about how it will age with the skin, ensuring clean, bold results that last.

About

Linework takes fine line tattooing to the extreme. It requires a very precise hand and clean design — every line is intentional. The result is elegant, delicate and highly detailed. Single needle technique allows for thin lines that create intricate compositions impossible with standard needles.
